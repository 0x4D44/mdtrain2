2. **No cab body** — `cab.ts` builds only a windscreen surround (4 bars + centre pillar) +
   desk + instruments. No enclosing shell (side walls/ceiling/floor/A-pillars), so the world
   shows all around the frame. Confirmed.

   reason clears only on `acknowledge` edge **AND at a stand**. Key is **Q** (help.ts row +
   keyboard). The "at a stand" requirement is NOT surfaced, and there's no on-screen prompt
   when a penalty fires (HUD has DRA/DSD/PENALTY/AWS *labels* only). Discoverability fix.
4. **Track straight** — `KINGSGATE_SEAHAVEN` has only 2 gentle curves (R=800 @4500-7000,
   R=500 @9200-10800) over 14 km; the rest is straight. Route-data change (pure, O19/O-RIBBON
   apply: ribbonHalfWidth 180 ⇒ minRadius ≥ 360, HLD floor 400).
5. **Sparse scenery** — `buildScenery`: ~360 trees (clumped), 2 overbridges, platform people;
   no bushes; buildings thin/absent. Density + variety fix (scenery.ts, instanced).
6. **OLE not staggered** — contact wire is a straight box at d=0 (no ±stagger to spread
   pan-head wear). (scene.ts buildContactWire.)
7. **OLE wire not connected** — wire at d=0; masts at ±mastD; no droppers/registration arms.
8. **OLE supports basic** — mast = one box post + one box arm. (scene.ts buildLineside.)
9. **Square raindrops** — rain is `THREE.Points` + `PointsMaterial` with no map ⇒ square
   point sprites. Confirmed in night/rain shot. (scene.ts rain.)
10. **Signals not visible** — only 3 starters (2120/5920/9920); head post/board are near-black
    (0x0c0f14/0x07090c), unlit lamps dark, so only the lit aspect shows a small dot that
    blends with the OLE mast. Read-poorly + sparse. (scene.ts buildSignals.)
11. **Stations float** — `buildStation` anchors the group at `formationHeight` (rail top);
    the ground beside the track is `terrainHeight ≈ formation+BALLAST_DROP(-0.4)` or lower on
    slopes, so the slab floats ≥0.4 m (worse in cuttings/embankments). Terrain-anchor fix.
