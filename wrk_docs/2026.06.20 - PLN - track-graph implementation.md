### 3.1 `sliceRoute` ŌĆö guarded sub-route extraction (DR/D-IMPL-2; pure; throws on a hazardous cut)

```ts
export function sliceRoute(route: Route, sFrom: number, sTo: number): Route;
```
Algorithm: assert `0 Ōēż sFrom < sTo Ōēż route.length`. **Cleanliness guard ŌĆö throw if any holds:**
`curvatureAt(route, sFrom) !== 0` or `curvatureAt(route, sTo) !== 0` (cut not on a ╬║=0 straight); any
station with `|chainage ŌłÆ cut| < platformHalf` for cut Ōłł {sFrom,sTo} (platform straddle); any signal
post or its derived reach `[chainageŌłÆAWS_MAGNET_OFFSET, chainage]` straddling a cut, OR a retained
signal whose `protects` station is not retained (orphaned cascade); any viaduct/tunnel band
`[centerŌłÆhalfLen, center+halfLen]` partially in [sFrom,sTo] (truncated band). Otherwise build:
- `length = sTo ŌłÆ sFrom`.
- each Segment band: keep segments overlapping [sFrom,sTo], clip `from=max(from,sFrom)`,
  `to=min(to,sTo)`, then subtract `sFrom` from from/to. (lookup is half-open + end-clamping ŌćÆ clean.)
- stations/signals: keep those with chainage Ōłł [sFrom,sTo); subtract sFrom from chainage.
- viaducts/tunnels: keep bands wholly inside; subtract sFrom from center. Drop `terrainSeed` (or keep).
Unit-tested: a clean slice round-trips (gradeAt/curvatureAt/speedLimitAt of the slice at s == original
at s+sFrom); a hazardous cut throws.
